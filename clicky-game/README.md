# Clicky Game
**Clicky Game** is a React based memory game application! 

# Instructions
Press each of the images to win, but be careful! Each image will shuffle after every click. Don't click one more than once!!

# Tech
**VS Code** - Text editor

**React** - Front-end design

**Node Js**
